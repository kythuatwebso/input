# input
Input Template


### Install
```
composer require websovn/input
```

### Method

```
->name(string $name)
->horizontal(boolean $horizontal)
->label(string $label)
->title(string $title)
->help(string $help)
->size(string $size)
->prepend(string $prepend)
->type(string $type)
->class(string $class)
->placeholder(string $placeholder)
->required(boolean $required)
->value(mixed $value)
->append(string $append)
```
